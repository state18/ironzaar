# IronZaar Challenge README

# Introduction

The IronZaar challenge is a high-difficulty challenge run for "The Bazaar," inspired by the renowned IronMon challenge. It emphasizes pure luck and strategic adaptation by stripping away player agency in pathing and upgrades.

# Rules

* All nodes are randomly selected using [https://www.random.org](https://www.random.org).  
* All level up options (Skill Vendors, Item/Loot/Gold, enchants, etc.) are randomly selected  
* All event options (Jules Cafe, Jungle Ruins, Tiny Furry Monster, etc.) are randomly selected  
* All enchant options are randomly selected  
* For any given node, generate 1-X, where X is the number of options in that event (Usually 1-3)  
* Your choice for Futura when running out of prestige the first time is randomly selected

## Node-specific exceptions

* Will refactor list above into this list as needed  
* If you roll an option you can’t afford, reroll until you can or leave (This mostly matters for Artisan Dunes and Gumball Machine event)  
* You may leave an event without doing anything if it would be harmful (Artisan Dunes, Mandala, etc.)  
* For the Gumball Machine Event, if you choose to buy any, generate a number 0-10 to decide how many to buy  
* For Aerodrome, you must roll 1-2 where 1 is reroll and 2 is select the location  
* For Lickit, first roll 1-2 to decide between the five treats or the creations, and then if you roll creations you roll 1-3 to decide which one  
* For Travel Agent you must buy both if you can, but if you can only afford one you must roll for it  
* Wink, Cult, and Expeditions must be selected when shown

## Additional rules for “Kaizo” mode

| Rule Name | Description |
| ----- | ----- |
| Tumbleweeds | The rightmost slot on your board must be empty during all combats |
| Steady Hand | You may not move existing items on your board and in your stash when inside of a node or before you roll to choose a node. |
| Mandala’s Malice | If the Mandala event is rolled, you may not change your board until it is over. |
| Highest-Rarity Tax | Starting from day 3, at the start of each day, you must sell an item of your choice of the highest rarity you have |
| No Repairs | Items that are destroyed during combat must be sold immediately after combat. |
| Take ONE Please\! | You may only buy 1 item per each batch of items shown at a shop. (So, if you reroll, you may buy 2 items total) |
| Digging for Gold | At Pearl’s Dig Site, you must roll 1-5 and dig for that many times before selecting your item. If you can’t afford it, you must leave the Dig Site without taking an item. |
